=БАЗЫ ДАННЫХ=

БД - набор данных, хранящихся в структурированном виде //https://habr.com/ru/articles/856576/
- каждый предмет имеет своё место и маркировку для быстрого поиска
- хранятся и обрабатываются в !элект.виде в компьютерной системе!

Система управления БД (СУБД) - система, позволяющая создавать БД и манипулировать сведениями из них

Взаимодействие с БД:
User хочет получить данные из БД -> формирует запрос на языке SQL -> СУБД принимает запрос и обращается к БД для поиска нужной ин-ции -> СУБД возвращает найденные данные user

-ТИПЫ БД-
1) Реляционные БД - данные организованы в виде связанных таблиц, для работы используют язык SQL (декларативный язык программирования, т.е. описывается ожидаемый рез-тат, а не способ его получения), с помощью него можно создавать/изменять объекты БД; запрашивать данные, фильтруя/сортируя/объединяя их; добавлять/обновлять/удалять записи; выполнять операции агрегации(sum, count, min, max, etc) и т.д.
ex.: <img width="447" height="239" alt="image" src="https://github.com/user-attachments/assets/d724db00-d364-45f0-9c62-faef251a84e2" />
a) PostgreSQL - бесплатная СУБД с открытым исходным кодом, отлично работает на linux и macos; требует следования стандартам SQL
б) MsSQL - СУБД от Microsoft, коммерческий продукт, родная платформа - windows; процедурный язык с большим кол-вом встроенных функций; простота написания сложных запросов
в) MySQL - бесплатная СУБД с открытым исходным кодом; версия языка в MYSQL близка к SQL; часто применяется в веб-разработке. MariaDB - ответвление от mysql, более новая и продвинутая версия

2) Нереляционные БД (NoSQL) - работа с любыми форматами данных: изображения, видео, аудио, JSON-документы ()
